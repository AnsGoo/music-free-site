# 反馈与参与

感谢你愿意帮助改进 MusicFree。请通过 [GitHub Issues](https://github.com/ansgoo/music-free-site/issues) 提交反馈。

## 提交前请先确认

1. **搜索已有 Issue**：避免重复提交。
2. **选择正确模板**（[新建 Issue](https://github.com/ansgoo/music-free-site/issues/new/choose)）：
   - **Bug 报告** — 程序异常
   - **文档问题** — 文档错误、链接失效
   - **功能特性** — 新功能或改进建议

## 功能建议

**[提交功能建议 →](https://github.com/ansgoo/music-free-site/issues/new?template=feature_request.yml)**

## Bug 报告

**[提交 Bug →](https://github.com/ansgoo/music-free-site/issues/new?template=bug_report.yml)**

## 文档问题

**[提交文档问题 →](https://github.com/ansgoo/music-free-site/issues/new?template=doc_report.yml)**

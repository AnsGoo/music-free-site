反馈与参与 ​

感谢你愿意帮助改进 MusicFree。请通过 GitHub Issues 提交反馈，便于跟踪与回复。

提交前请先确认 ​

  1. 搜索已有 Issue：避免重复提交相同问题或建议。
  2. 区分反馈类型：
    • 程序 Bug / 功能需求：MusicFree 服务端、管理后台、插件宿主等行为问题。
    • 文档问题：本站说明错误、链接失效、示例过时等，可在 Issue 中注明「文档」。
  3. 尽量提供可复现信息：版本号、操作步骤、日志或截图能显著加快处理速度。

功能建议 ​

有新的功能想法或体验改进？请使用 功能特性 模板提交：

提交功能建议 →

建议在描述中说明：

  • 使用场景与要解决的问题
  • 期望的功能行为
  • 可选：参考产品、截图或相关链接

Bug 报告 ​

遇到异常行为或报错？请使用 Bug 报告 模板提交：

提交 Bug →

请尽量填写：

信息说明
MusicFree 版本镜像标签、发布版本或 commit
复现步骤从操作到出现问题的完整路径
实际 vs 期望当前结果与你认为正确的结果
环境部署方式（Docker / 裸机）、操作系统、相关插件

其它参与方式 ​

  • 浏览全部 Issue：github.com/ansgoo/music-free-site/issues
  • 文档改进：欢迎直接对本仓库提交 Pull Request
  • 插件相关：插件 Bug 或功能可在对应插件仓库反馈；注册表与开发说明见 插件开发
